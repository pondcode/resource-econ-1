STRICT LATEX GRAPH RULES FOR MY CALCULATOR

My calculator can render this format:

$$
\begin{array}{l}
...
\end{array}
$$

Use this format for graph mini-guides.

Very important:
- Do NOT use normal spaces for positioning inside LaTeX arrays.
- Normal spaces may collapse or fail.
- Use \quad for spacing.
- Keep each line short.
- Do NOT use complicated graph drawings.
- Do NOT use TikZ.
- Do NOT use PGFPlots.
- Do NOT use \hspace.
- Do NOT use \qquad unless necessary.
- Prefer \quad.
- Do NOT redraw the full graph in every step.
- Each step must show ONLY the new curve, line, label, or shading instruction.
- The final graph may be rough, but the written description must be very specific.

For every graph step, use this exact structure:

Step X: Add [new item]

Description:
Explain in normal text where I draw it by hand.

LaTeX guide:
$$
\begin{array}{l}
New\ curve:\\
|\quad\quad/\\
|\quad/\\
|/
\end{array}
$$

Do not put the whole explanation inside LaTeX.
Use LaTeX only for the small guide and equations.
